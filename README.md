# fitcheck

Voice-based outfit recommendation app - your best fit, every time.

## Design

This UI was created from a Figma design featuring:
- Clean, minimalist black interface
- Voice-first interaction pattern
- Example prompts to guide users
- Interactive microphone button with visual feedback

## Tech Stack

- **React 18** - UI framework
- **TypeScript** - Type safety
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **Lucide React** - Icons

## Getting Started

### Install dependencies
```bash
npm install
```

### Run development server
```bash
npm run dev
```

### Build for production
```bash
npm run build
```

## Features

- Responsive design optimized for mobile (iPhone 16 Pro)
- Interactive microphone button with listening state
- Hover effects on suggestion prompts
- Smooth animations and transitions
- Clean, accessible interface

## Usage

1. Tap the microphone button to start voice input
2. Say your occasion (e.g., "Going to a dinner date")
3. Receive personalized outfit recommendations
